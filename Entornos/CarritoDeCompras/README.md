# Práctica de Refactorización

## Descripción

Este repositorio contiene los ejercicios correspondientes a la práctica de refactorización para la asignatura de Entornos de Desarrollo. El objetivo es mejorar la calidad, legibilidad y mantenibilidad del código existente aplicando técnicas de refactorización.

El enunciado completo del ejercicio se encuentra en el archivo `enunciado.txt`.

## Objetivos

- Identificar y eliminar código duplicado.
- Mejorar la estructura y organización del código.
- Aplicar principios SOLID y buenas prácticas.
- Incrementar la legibilidad y comprensión del código.

## Estructura del repositorio

- `CarritoDeCompras.java`: Clase que representa un carrito de compras, maneja una colección de items con sus cantidades.
- `Item.java`: Clase que representa un ítem en el carrito.
- `Main.java`: Clase principal que demuestra el uso del carrito de compras.
- `enunciado.txt`: Archivo que contiene el enunciado completo del ejercicio de refactorización.
- `README.md`: Este archivo de documentación.

## Instrucciones

1. Analiza el código proporcionado.
2. Identifica posibles mejoras y aplica refactorizaciones.
3. Documenta los cambios realizados.
4. Ejecuta las pruebas para asegurar que todo funciona correctamente.
